Integration Guide

To integrate:

  1. Generate an API key from the dashboard.
  2. Add the script tag: <script src="https://yourdomain.com/api/client_script/YOUR_API_KEY"></script>
  3. The script auto-integrates DDoS protection and adds a footer: "Protected And Supported By SolDev Security".
  4. Users solve a PoW CAPTCHA when rate limits are hit.
  5. Valid tokens allow access to protected endpoints.

Malicious traffic is redirected to an invalid domain and IPs are blacklisted. The system handles heavy loads via threading and queuing.
